# Dark Skies Database

## Running the server:
1. Clone this repo:
   
   ```
   git clone git@github.com:LukeTHardy/dark-skies-api.git
   ```
2. Install JSON-server:
   
   ```
   npm install json-server
   ```
3. Navigate to repo directory:
   
   ```
   cd dark-skies-api
   ```
4. Start the server:
   
   ```
   json-server -w database.json -p 8088
   ```
